• ASP Linux
    A 686 family (PIII, Celeron,etc) optimized distro out of Moscow (Russia, not Idaho). They claim to be 100% RH compatible, so using any RH RPM should ensure compatibility.
  • Debian GNU/Linux
  • FloppyFW
    Screening router/Package filtering firewall all on a 486 with no hard drive. For the truly frugal
  • Immunix
    Security hardened Linux distribution.
  • LinuxISO
    Bootable CD images for the most popular Linux distributions.
  • Linux from Scratch
  • Mandrake Linux
  • Netule Linux
    A series of specialized Linux distributions with a common base system.
  • RedHat Linux
  • RULE (Run Up2date Linux Everywhere) project
  • Slackware Linux
    Aims to be a Simple, Stable, Unix-like distribution
  • SuSE Linux Portal Knowledge
  • Turbolinux
    Aimed at the large server market including Intel's new Itanium chip.
  • Ultralinux
    Starting place to learn about Linux ports to machines running on the Sparc chip, as well as other Sun boxes.
  • Yellow Dog Linux
    Distribution for PowerPC computers.

A more complete list of distributions can be found on the Linux Weekly News Distributions Web page.
