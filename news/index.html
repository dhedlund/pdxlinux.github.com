Web page news

17 January 2003 After some difficulties, we have recovered the original content of this site. Discussion is underway concerning reversion of the new format.

30 July 2002 David Mandel has contributed a Short History of PLUG.

20 July 2002 Karl Hegbloom, sysadmin for Drizzle, the server PLUG's web page lives on, added a collection of web-accessible documentation. (Note: this is the old web page, and not all links on that page work.)

16 July 2002 At the request of Zot O'Connor, information about the Linux Advanced Topics Talk was added to the web site.

18 June 2002 Geoff Burling revised the content of the main page, and the first Resources page, as well as added more meeting information.

22 May 2002 Wil added a link to the #orlug IRC channel home page.

8 May 2002 Wil Cooley re-implemented the site using a very simple, home-brewed PHP template system.

28 March 2001 Comment page added.

7 January 2001 Daniel Hedlund (who created much of the webpage design) created a mailing list to discuss topics/opinions/comments relating to the PLUG website.
